## 실행 방법

각 프로젝트 디렉터리로 이동하여 의존성을 설치하고 실행 스크립트를 사용하세요.

예시:

```bash
cd lecture08-book
npm install
npm run dev
```

(각 프로젝트의 `package.json`을 참고하여 패키지 매니저 및 스크립트를 확인하세요.)
